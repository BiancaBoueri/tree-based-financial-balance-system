# Tree-Based Financial Balance System
API backend desenvolvida em Go para processamento de balanço financeiro hierárquico utilizando estruturas em árvore.

Este projeto é uma reimplementação de um projeto antigo originalmente desenvolvido em TypeScript, agora reconstruído com foco em aprendizado de backend com Go e organização de APIs.

## Objetivos

- Aprender desenvolvimento backend com Go
- Implementar autenticação JWT
- Trabalhar com estruturas hierárquicas em árvore
- Processar balanços financeiros
- Construir uma API REST simples e organizada

## Tecnologias

- Go
- SQLite
- JWT
- net/http
